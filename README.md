# SourceREACH
De-lensing source reconstruction code with adaptive smoothing, developed for the paper, "SourceREACH (Source REconstruction of Arcs behind Cluster Halos): A New Source Reconstruction Algorithm Optimized for Giant Arcs and Galaxy Cluster Lenses" (Eid &amp; Keeton, 2026).  Developed using Hubble Frontier Fields data for Abell 370.
